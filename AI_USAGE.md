# AI Usage Disclosure

This project was developed with assistance from Microsoft Copilot.  
All architectural decisions, code reviews, and final implementations were performed and validated by a human maintainer.

## What AI Was Used For
- Code generation and refactoring
- Linting and static analysis improvements
- Documentation drafting
- CLI ergonomics and UX design
- Metadata schema design
- Bug triage and reproduction steps

## What AI Was *Not* Used For
- Storing or processing sensitive data
- Providing API keys or secrets
- Making licensing decisions
- Releasing unreviewed code

## Session Logs
Major AI-assisted development sessions are summarized in this directory.  
Raw transcripts are not stored to avoid leaking sensitive paths or tokens.

Each summary includes:
- Date
- Purpose
- Commands generated
- Files modified
- Human review notes
