# Contributing Guidelines

Thank you for your interest in contributing!

## Code of Conduct
By participating, you agree to uphold the standards of respectful, constructive collaboration.

## How to Contribute
- Open an issue before submitting large changes.
- Use feature branches (`feature/...`, `fix/...`).
- Ensure linting and Semgrep checks pass.
- Add or update tests when modifying logic.

## AI-Assisted Contributions
AI tools may be used, but contributors must:
- Disclose AI assistance in the PR description.
- Add a commit trailer: `Assisted-by: AI`
- Verify and validate all generated code.
- Ensure no sensitive data or secrets appear in prompts or outputs.

## Development Setup
- Node version: (add your version)
- Install dependencies: `npm install`
- Run lint: `npm run lint`
- Run tests: `npm test`

## Security
Do not include API keys, tokens, or secrets in issues, PRs, or commits.
